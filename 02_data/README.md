# 数据集说明文档
本目录存放地下水Sim2Real渗透系数反演项目的所有数据集、元数据与标准化器。

## 目录结构
02_data/
├── 01_raw_sim_data/ # 原始仿真数据集
└── 02_metadata/ # 元数据、标准化器、数据描述文件

## 1. 原始仿真数据集
### 数据集基本信息
| 项目 | 内容 |
|------|------|
| 文件名 | sim2real_full_dataset.npz |
| 生成时间 | 2026-05-09 09:46:56 |
| 总样本量 | 1000组 |
| 有效样本量 | 1000组 |
| 随机种子 | 42 |

| 生成工具 | MODFLOW 6.7.0 + FloPy 3.10.0 |

### 数据集结构
| 数组名 | 形状 | 含义 |
|--------|------|------|
| head | (1000, 5) | 模型输入：5口监测井的稳态水头值，单位m |
| k | (1000, 100) | 模型标签：10×10网格的渗透系数场（展平），单位m/d |
| monitor_wells | (5, 3) | 监测井的空间坐标 |
| grid_size | (2,) | 模型网格尺寸 |
| random_seed | 标量 | 随机种子，保证实验100%可复现 |
| dataset_stats | dict | 数据集统计信息 |

### 水文地质模型设置
- 网格尺寸：10行×10列×1层，网格步长100.0m×100.0m
- 含水层高程范围：0.0~100.0m
- 边界条件：左右两侧定水头（左100m，右90m）
- 模拟类型：稳态模拟
- 渗透系数范围：1.0~5.0 m/d
- 监测井数量：5口，位置：`[[0, 2, 2], [0, 2, 7], [0, 5, 5], [0, 7, 2], [0, 7, 7]]`

## 2. 元数据说明
`02_metadata/` 目录存放：
- 数据标准化器：`data_scalers_*.npy`，包含训练集的均值、标准差，用于数据归一化/反归一化
- 数据集元数据：`dataset_metadata.json`，包含数据集完整的统计信息、生成配置、运行环境
- 数据集描述文档：`dataset_description.md`，可直接用于论文方法部分

## 3. 数据使用说明
1.  训练代码会自动从本目录加载数据集与标准化器
2.  数据集文件超过100MB，未纳入Git版本管理，可通过项目根目录README中的Zenodo/网盘链接下载
3.  所有数据生成代码见 `01_code/01_data_generation/` 目录，可100%复现数据集生成过程
