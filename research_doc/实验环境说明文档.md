# 水文地质参数反演实验环境说明文档

## 1. 硬件环境
- CPU型号：AMD 64 Family 25 Model 97 Stepping 2 AuthenticAMD ~2401 Mhz（锐龙9 7940HS）
- 核心数：8核16线程
- 内存容量：16.0 GB
- GPU型号：NVIDIA GeForce RTX 5060 Laptop
- 显存：8GB（8151MiB）
- CUDA驱动版本：13.1（Driver Version 592.01）
- cuDNN版本：未安装（当前为PyTorch CPU版本，无GPU加速需求）

## 2. 操作系统
- 系统版本：Windows 11 家庭中文版 25H2 (OS 内部版本 26200.8246)

## 3. Python运行环境
- 虚拟环境名称：hydro_ai
- Python版本：3.10.20

## 4. 核心依赖版本
- PyTorch版本：2.2.0
- PyTorch绑定CUDA版本：None（CPU版本，无CUDA支持）
- numpy版本：1.26.4
- matplotlib版本：3.8.4
- pandas版本：2.2.1

## 5. 依赖安装说明
本实验的所有Python依赖版本已固化，可通过以下命令一键复现环境：
```bash
# 1. 创建虚拟环境
conda create -n hydro_ai python=3.10 -y
conda activate hydro_ai

# 2. 安装依赖
pip install -r requirements.txt
