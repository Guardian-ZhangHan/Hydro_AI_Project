# Hydro Inversion Model Card
## 1. 模型基本信息
| 项目 | 内容 |
|------|------|
| 模型名称 | Sim2Real Permeability Inversion MLP |
| 版本 | v1.0（Day5 Baseline） |
| 任务 | 基于监测井水头数据的二维潜水含水层渗透系数场反演 |
| 适用场景 | 地下水Sim2Real仿真到真实迁移学习，水文地质参数反演，地下水流数值模拟 |
| 开发机构 | 中国地质大学（武汉）环境学院 |
| 开源协议 | MIT |

## 2. 模型架构
| 配置项 | 参数 |
|--------|------|
| 输入维度 | 5（5口监测井的稳态水头观测值） |
| 输出维度 | 100（10×10网格的渗透系数场展平结果） |
| 隐藏层结构 | [64, 128, 256, 128] |
| 激活函数 | ReLU |
| 归一化层 | BatchNorm1d |
| Dropout | 0.0（基线版，无正则） |
| 总参数量 | 88,676 |
| 可训练参数量 | 88,676 |

## 3. 训练配置
| 配置项 | 参数 |
|--------|------|
| 优化器 | Adam |
| 初始学习率 | 1e-3 |
| 权重衰减（L2正则） | 1e-5 |
| 损失函数 | 均方误差（MSE） |
| 学习率调度器 | ReduceLROnPlateau，衰减系数0.5，耐心轮次10 |
| 早停机制 | 连续150轮验证损失无改善则终止训练，自动恢复最优权重 |
| 最大训练轮次 | 300 |
| 批次大小 | 32 |
| 随机种子 | 42（全链路固定，包含数据划分、模型初始化、CUDA操作） |
| 训练设备 | CPU/GPU自适应 |

## 4. 性能指标（仿真测试集）
测试集为100组独立的MODFLOW仿真样本，未参与训练与验证过程。
| 评估指标 | 数值 | 单位 |
|----------|------|------|
| 均方误差（MSE） | 1.0747 | - |
| 根均方误差（RMSE） | 1.0367 | m/d |
| 平均绝对误差（MAE） | 0.8788 | m/d |
| 决定系数（R²） | -0.0666 | - |

### 物理合理性校验
所有反演得到的渗透系数值均处于0.5~5.5 m/d的水文地质合理区间内，无负值、超出边界范围的异常值，完全符合地下水动力学基本规律。

## 5. 训练数据集
- 数据集名称：sim2real_full_dataset.npz
- 样本量：1000组二维潜水含水层稳态流仿真样本
- 数据划分：训练集800组、验证集100组、测试集100组
- 详细数据集说明见 `02_data/README.md`

## 6. 使用限制与注意事项
1.  当前版本为仿真数据集训练的基线模型，尚未进行野外真实数据的微调与域适配，直接用于真实场景会存在较大误差
2.  模型仅适用于稳态流、单含水层、左右定水头边界的简化水文地质模型
3.  模型输入为5口固定位置监测井的稳态水头值，更换监测井位置需重新训练模型
4.  本模型仅用于科研与教学用途，不构成任何工程水文地质勘察的决策依据
