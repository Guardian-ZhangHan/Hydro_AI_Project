# Day4 学习日志
## 一、今日学习目标
1. 理解欠拟合、过拟合核心定义，以及少样本水文反演场景下的成因与危害
2. 复现少样本条件下典型过拟合现象：训练损失下降、验证损失上升
3. 掌握三种工业级防过拟合方法：L2正则化、Dropout、早停法
4. 组合三种方法优化模型，抑制过拟合、提升泛化能力，输出可论文使用的实验结果

## 二、实操完成步骤
1. 新建 learning_day4 文件夹，保持每日独立归档规范
2. 构造极少量训练样本+独立验证样本，模拟野外水文数据稀缺场景
3. 搭建基础模型，先复现无防护下的标准过拟合现象
4. 依次加入：Dropout层、L2权重衰减、早停法策略
5. 运行训练，自动判定早停触发轮次，量化过拟合抑制效果
6. 生成训练/验证损失曲线、导出损失数据CSV，整理实验分析结论

## 三、核心原理收获
1. 地下水参数反演属于典型少样本任务，极易发生过拟合
2. 过拟合表现为训练集拟合极好、验证集泛化差，无法用于真实场地预测
3. Dropout随机失活神经元、L2约束权重幅值、早停防止过度训练，三者组合效果最优

## 五、遇到问题与解决办法
1. 问题：训练循环重复编写，导致损失列表长度和绘图维度不匹配报错
  解决：精简合并单一训练循环，损失记录跟随实际训练轮数
2. 问题：Dropout添加后未在前向传播中调用，未生效
  解决：在隐藏层激活后插入Dropout，训练模式自动启用、验证模式自动关闭
3. 问题：早停逻辑位置放错，判断失效
  解决：放在每轮验证损失计算完成后，再执行早停判定与最优模型保存
