# Day1 学习日志
## 一、今日学习目标
1. 搭建地下水渗透系数反演基础3层全连接神经网络骨架
2. 掌握网络输入层、隐藏层、输出层维度设计逻辑，匹配5口监测井→10×10网格K值反演场景
3. 完成Tanh、ReLU两种激活函数代码实现与输出结果对照实验
4. 固定全局随机种子，保证实验可复现，符合学术研究控制变量规范

## 二、实操完成步骤
1. 按照项目规范，新建 learning_day1 独立文件夹，统一归档当日所有代码与数据
2. 导入PyTorch、NumPy等依赖库，设置固定随机种子，杜绝实验结果随机波动
3. 搭建HydroInversionNN网络结构：输入5维、隐藏层32/64神经元、输出100维
4. 分别引入Tanh、ReLU激活函数，编写前向传播逻辑
5. 生成模拟水头输入数据，分别运行两种激活函数，导出输出结果CSV文件
6. 对比两种激活函数输出差异，保存差值统计文件，完成实验归档

## 三、核心原理收获
1. 全连接神经网络适合一维监测井水头到网格K值的映射回归任务
2. 激活函数决定网络非线性拟合能力，Tanh输出正负对称、分布连续，更适配水文参数空间分布特征；ReLU易出现梯度消失、输出分布偏置
3. 学术实验必须固定随机种子，保证模型初始化、数据生成完全可复现，是论文结果可信的前提

- 核心代码：网络骨架搭建代码、激活函数对比代码
- 产出文件：relu_output.csv、tanh_output.csv、activation_abs_diff.csv

## 五、遇到问题与解决办法
1. 问题：未固定随机种子，每次运行输出结果不一致
  解决：统一设置SEED=42，全局固定CPU、GPU随机种子
2. 问题：网络维度和实际反演场景不匹配
  解决：严格按照5口监测井输入、10×10网格输出设计网络维度
