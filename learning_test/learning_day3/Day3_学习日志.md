# Day3 学习日志
## 一、今日学习目标
1. 理解梯度下降、反向传播、Adam优化器核心原理
2. 掌握深度学习模型完整训练闭环：前向传播→损失计算→反向传播→参数更新
3. 设计3组学习率对照实验，用控制变量法筛选水文反演最优学习率
4. 自动生成损失曲线与实验数据，满足论文图表使用要求

## 二、实操完成步骤
1. 新建 learning_day3 文件夹，沿用项目统一目录规范
2. 复用Day1网络骨架、Day2 MSE损失函数，保证实验变量唯一
3. 引入Adam优化器，设置3组对照学习率：1e-2、1e-3、1e-4
4. 固定模型结构、随机种子、训练轮数等所有条件，仅学习率为变量
5. 执行训练，记录每轮损失变化，自动绘制多组损失收敛曲线
6. 导出各学习率损失历史CSV、高清论文级曲线图，量化分析收敛效果

## 三、核心原理收获
1. 反向传播依托链式法则求导，计算每一层权重梯度
2. Adam具备自适应学习率、动量加速收敛，是水文少样本反演首选优化器
3. 学习率过大会震荡不收敛，过小会收敛过慢，必须通过对照实验优选

1. 问题：学习率1e-2损失剧烈震荡
  解决：对照实验验证该学习率不适用于本反演场景，淘汰该参数
2. 问题：不会批量记录损失并绘图
  解决：建立字典存储各组损失，循环绘图并统一保存高清格式
