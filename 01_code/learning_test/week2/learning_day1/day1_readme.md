
## ⚙️ 数据处理流程
1.  **数据读取**：加载 `full_dataset.npz` 中的 `head_raw` 原始水头数据
    - 原始数据形状：`(1, 100)`
    - 数据类型：`float64`
2.  **无效值清洗**：将MODFLOW中的NaN/inf值替换为0.0
3.  **张量转换**：NumPy数组 → PyTorch CPU张量（`float32`）
4.  **维度变换**：适配AI模型输入格式，增加通道维度
    - 变换前：`(1, 100)`
    - 变换后：`(1, 1, 100)`
5.  **标准化处理**：0~1归一化，消除量纲影响
6.  **结果保存**：输出 `day1_final_tensor.pt`，可直接用于后续模型训练

## ✅ 关键结果
- 最终张量形状：`torch.Size([1, 1, 100])`
- 运行设备：`cpu`
- 数值范围：`0.0000 ~ 1.0000`
- 无任何报错，全流程可复现
