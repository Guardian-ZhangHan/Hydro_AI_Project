# Week2-Day2：张量验证与预处理操作
## 任务目标
加载Day1标准化张量，完成验证、分析、切片、维度变换、格式转换，构建论文级可复现数据流程。

## 输入文件
- day1_final_tensor.pt （Day1输出）

## 输出文件
- day2_processed_tensor.pt （Day2处理后张量）

## 核心操作
1. 张量加载与完整性验证
2. 统计量分析（最小/最大/均值/标准差）
3. 张量切片（前50特征提取）
4. 维度增减（squeeze / unsqueeze）
5. 张量 ↔ numpy 双向转换
6. 结果保存

## 验证结果
- 输入形状：[1, 1, 100]
- 数值范围：[0,1]
- 无异常值、无NaN、无报错
- 流程可复现、符合论文数据预处理规范
