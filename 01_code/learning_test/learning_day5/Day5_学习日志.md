## 三、核心原理收获
1.  水文反演场景下，Sim2Real是解决少监测数据反演痛点的核心范式，通过仿真数据预训练+真实数据微调，大幅降低对真实数据的依赖
2.  数据集预处理的核心学术规范：必须仅用训练集拟合归一化器，避免数据泄露，确保模型泛化性能评估的客观性
3.  反归一化一致性是模型结果可解释的关键，误差必须≤1e-6，才能保证反演结果可还原为真实水文地质参数
4.  DataLoader批量加载器是PyTorch模型训练的基础，批量大小与数据形状必须严格匹配，避免训练中断
